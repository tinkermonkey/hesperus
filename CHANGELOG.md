## [0.1.0] - 2026-02-23

### Added
- Initial release of Hesperus theme
- Complete Flowbite React theme with 60+ styled components
- Dark mode support with automatic theme switching
- Retro CRT aesthetic with grid backgrounds and noise filters
- Monospace typography with Space Mono font
- Custom color system with CSS custom properties
- Interactive component browser for development
- TypeScript declarations
- Comprehensive documentation

### Components Themed
- **Core**: Button, Badge, Card, Modal
- **Forms**: TextInput, Select, Checkbox, Radio, FileInput, Label, ToggleSwitch
- **Navigation**: Navbar, Sidebar, Breadcrumb, Tabs, Pagination
- **Data Display**: Table, List, Avatar, Timeline, Accordion
- **Feedback**: Alert, Toast, Spinner, Progress, Rating
- **Content**: Typography, HR, Blockquote, Kbd
- **Overlays**: Tooltip, Dropdown, Popover, Banner
- **Layout**: ButtonGroup, Footer
